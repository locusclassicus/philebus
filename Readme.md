## Об этом проекте

Репозиторий содержит комментарий к диалогу "Филеб" и новый перевод этого диалога, выполненные О.В. Алиевой. 
Авторские права на все материалы принадлежат О.В. Алиевой. Перевод и комментарий носят предварительный характер. 

В 2024-2025 г. исследование проводилось за счет гранта Российского научного фонда № 24-28-00222, https://rscf.ru/project/24-28-00222/

Ссылка для цитирования формируется динамически, см. информацию на странице "О проекте".
